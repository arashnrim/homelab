# Umami

Umami is a simple, fast, privacy-focused alternative to Google Analytics. It's a free and open-source web analytics solution that respects user privacy while providing you with the insights you need to understand your website traffic.

# Things to note

- The environment variable files is incomplete; make sure to complete them (or remove any irrelevant fields) before running.
  - Under `.env`, ensure that `APP_SECRET` is an alpahnumeric string. This is used internally by Umami.
