# Actual Budget

Actual Budget is a self-hosted personal finance and budgeting application. It helps you manage your finances locally with full control over your data.

## Things to note

- No environment variables are required for basic operation.
- The web interface is available at port `9002` (mapped to the app's port 5006).
- Health checks are configured to ensure the application is running properly.
- All data is persisted in a Docker volume for data persistence and backup.
